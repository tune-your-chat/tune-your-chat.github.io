Matrix is a decentralised protocol: like email, you need to have an account provider to start using matrix.

But picking a provider can be overwhelming for your first steps. The Matrix.org Foundation is hosting the matrix.org Matrix server for every newcomer to create an account. You can safely use it to create an account when getting started.
